# Navjeevan Educational Website System

This repository contains the frontend architecture for the Rajasthan Relief Society / Navjeevan education websites.

## Sites

- Main domain landing page: `navjeevanvidhalya.in`
- Navjeevan Junior College: `navjeevanjuniorcollege.navjeevanvidhalya.in`
- Bajaj Degree College: `bajajdegreecollege.navjeevanvidhalya.in`

## Local Folders

- `index.php`, `css/`, `images/`: main domain landing page
- `juniorCollege/`: Navjeevan Junior College source
- `bajajdegreecollege/`: Bajaj Degree College source
- `navjeevan_project_context/`: project handoff notes

## Context Files

- `PROJECT_CONTEXT.md`: architecture, design, domains, admissions, and current decisions
- `TODO.md`: completed work and remaining tasks
- `DEPLOYMENT.md`: cPanel upload mapping, HTTPS, `.htaccess`, and deployment checks

## Stack

- PHP includes
- HTML/CSS
- Vanilla JavaScript
- Responsive design
- Google Forms for Junior College admissions
- Linux cPanel hosting

## Key Features

- Main institutional landing page
- Separate source folders for both college subdomains
- Reusable college layouts
- Responsive navbar
- Mobile-first behavior
- Dynamic page titles
- Legal pages
- Gallery and contact pages
- Lightweight architecture without WordPress

## Goal

A secure, maintainable, low-cost educational website system without WordPress dependency.
